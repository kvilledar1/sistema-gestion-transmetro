# Sistema de Gestión del Transmetro

## Descripción
Sistema web desarrollado para la administración y control operativo del Transmetro.

## Funcionalidades
- Gestión de líneas
- Gestión de estaciones
- Gestión de accesos
- Gestión de buses
- Gestión de pilotos
- Gestión de parqueos
- Gestión de operadores
- Registro de incidencias
- Generación de reportes
- Administración de usuarios
- Configuración del sistema

## Tecnologías utilizadas
- PHP
- HTML5
- CSS3
- MySQL
- XAMPP
